Hướng dẫn cài đặt chương trình
  

 

Bước 1: Cài đặt môi trường phát triển ứng dụng.

-         Cài đặt JDK tại:Do Van Tien - Nguyen Phuoc Cuong\Tools\Lib

-         Cài đặt Eclipse tại: Do Van Tien - Nguyen Phuoc Cuong\Tools\eclipse

-         Cài đặt Server MySQL tại: Do Van Tien - Nguyen Phuoc Cuong\Tools\MySQL.

Lưu ý: chương trình có sử dụng hibernate để kết nối với cơ sở dữ liệu sử dụng tên và mật khẩu mặc định là :

User:root

Pass: root

Do đó trong quá trình cài đặt nếu hai thông tin chưa phù hợp với những cài đặt trên, người dùng phải config lại thông tin cho phù hợp tránh phải vào code chỉnh sửa.

Bước 2: Restore Database

Trong dvd có database sẵn của chương trình, khi đã cài đặt server người dùng thực hiện các bước sau để lấy

Hình 1: Tạo mới một database tên dbsa

Hình 2 : Restore dbsaDatabase dbsa có trong dvd vào database dbsa vừa tạo

Lưu ý: vì file script chứ một số lượng lớn dữ liệu do đó thời gian restore khoảng 15 đến 20 máy tùy thuộc vào cấu hình máy.

Bước 3:Compile source code chương trình.

Sử dụng Eclipse mở project trong source.

Lưu ý: Cần add các thư viện cần thiết nếu như compiler báo thiếu thư viện, các thư viện có trong thư mục lib của source code/

Hình 3: Add thư viện cần thiết cho chương trình.

Bước 4: Chạy chương trình

Thực hiên chạy chương trình, trước khi chạy nên check kết nối của hệ thống với Internet và kết nối với database (đã có chức năng trong chương trình).
